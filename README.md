Στην εταιρεία που εργάζεστε ως προγραμματιστής σας ζητήθηκε να υλοποιήσετε
ένα πρόγραμμα που να υπολογίζει τα ελάχιστα εναπομείναντα έτη μέχρι τη
συνταξιοδότηση ενός υπαλλήλου, καθώς και τα συνολικά έτη εργασίας του.
Ως είσοδοι στο πρόγραμμα ζητούνται τα εξής:
❏ Το όνομα του υπαλλήλου.
❏ Το επώνυμο του υπαλλήλου.
❏ Το έτος γέννησής του.
❏ Το έτος κατά το οποίο εκκίνησε την εργασία του.
Έπειτα το πρόγραμμα θα πρέπει να υπολογίζει τα εξής:
❏ Τα ελάχιστα έτη που απαιτούνται για τη συνταξιοδότησή του (μεταβλητή
min_years) δεδομένου πως μπορεί να πάρει σύνταξη όταν ολοκληρώσει
το 65ο έτος της ηλικίας του. Ο υπολογισμός θα γίνει με βάσει το τωρινό
έτος (2023).
❏ Τα συνολικά έτη κατά τα οποία έχει εργαστεί όταν και θα πάρει σύνταξη
(μεταβλητή worked_years)
